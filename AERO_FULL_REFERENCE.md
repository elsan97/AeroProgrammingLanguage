# Aero Language V4 - Полный справочник

## 🎯 ОСНОВЫ ЯЗЫКА

### Синтаксис
```
{Aero}{progname}Название{|}{progcode}Код программы{|}{|}
{game}{gamename}Название{|}{gamecode}Код игры{|}{|}
```

### Структура программы
```aer
{Aero}{progname}Моя программа{|}{progcode}
{charset}UTF-8{|}
{echo}"Привет, мир!"{|}
{|}
```

---

## 📚 БАЗОВЫЕ КОМАНДЫ

### Структура программы
| Команда | Описание |
|---------|----------|
| `{Aero}` | Начало программы |
| `{progname}Название{|}` | Название окна программы |
| `{progcode}Код{|}{|}` | Код программы (создаёт окно) |
| `{progico}путь{|}` | Иконка программы |
| `{game}` | Начало игры (полноэкранный режим) |
| `{gamename}Название{|}` | Название игры |
| `{gamecode}Код{|}{|}` | Код игры |
| `{gameico}путь{|}` | Иконка игры |
| `{charset}кодировка{|}` | Установить кодировку |

### Кодировки для {charset}
| Кодировка | Описание |
|-----------|----------|
| `UTF-8` | Unicode (по умолчанию) |
| `CP1251` | Windows кириллица |
| `CP866` | DOS кириллица |
| `KOI8-R` | Unix кириллица |
| `ISO-8859-5` | ISO кириллица |
| `ASCII` | Английский |

### Вывод
| Команда | Пример | Описание |
|---------|--------|----------|
| `{echo}текст{|}` | `{echo}"Привет"{|}` | Вывод текста |
| `{print}текст{|}` | `{print}"Текст"{|}` | Вывод без \n |
| `{clear}{|}` | `{clear}{|}` | Очистить вывод |
| `{pause}{|}` | `{pause}{|}` | Пауза |

### Ввод
| Команда | Пример | Описание |
|---------|--------|----------|
| `{input}#var{|}` | `{input}#name{|}` | Ввод с клавиатуры |
